`egg-echat` 是一个基于 `eggjs `框架的 私有化部署版-企业微信 **非官方** SDK。

`egg-echat` 的安装非常简单，由于它是一个标准的 `egg-plugin` 包，这意味着，可以很方便的在 `eggjs` 中应用。当然在其他框架也可以用，`echat` 标准的  `npm` 包正在开发中...。

> 当前版本： 1.0.8
